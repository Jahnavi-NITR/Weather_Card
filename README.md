# Weather_Card

A simple and responsive Weather Card built with HTML, CSS and JavaScript that fetches real-time weather data using the OpenWeatherMap API.

## Live Demo

Demo Link: **https://jahnavi-nitr.github.io/Weather_Card/**

## Features
 
- **Search weather by city name**
- **Real-time temperature display**
- **Humidity and wind speed details**

## Built With
 
- **HTML** — page structure and weather card layout
- **CSS** — responsive styling, gradients
- **JavaScript** — API fetching, DOM updates, events
- **OpenWeatherMap API** — live weather data
